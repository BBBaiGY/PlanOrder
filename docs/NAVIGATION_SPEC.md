# 导航栏规范文档

本文档记录了导航栏的固定配置要求，后续修改应尽量避免改动这些配置。

## 颜色规范

### 背景色
- **导航栏背景**: `rgba(0, 0, 0, 0.85)`
- **选中项背景**: `#5c5c5c`
- **悬停背景**: `rgba(255, 255, 255, 0.1)`

### 文字颜色
- **非选中状态**: `#fff` (白色)
- **选中状态**: `#02b980` (绿色)
- **系统主色调绿色**: `#02b980`

### 链接颜色
- **链接颜色**: `#1890ff` (蓝色)

## 布局规范

### Logo区域
- **Logo图标**: 工业制造风格齿轮图标（与标题同高）
- **Logo图标容器**: 24px × 24px（与标题行高一致）
- **Logo文字**: "小工单产品原型"
- **Logo文字大小**: 16px
- **Logo文字行高**: 1.5

### 菜单结构
1. 主页
2. 销售管理
3. **需求管理** (包含：供需检查)
4. 采购管理
5. 生产管理
6. 委外管理
7. 质量管理
8. 上下游管理
9. 库存管理
10. 任务管理
11. 基础数据
12. 自定义配置
13. 报表
14. 系统配置
15. 车间看板
16. 自定义报表
17. 智能看板
18. 插件市场

### 菜单样式
- **选中效果**: 
  - 背景色: `#5c5c5c`
  - 文字颜色: `#02b980`
  - 右侧圆角: `border-radius: 0 8px 8px 0`
  - 无左边框
- **二级菜单对齐**: 文字与一级菜单文字对齐（padding-left: 40px）

### 搜索框
- **占位符**: "搜索(⌘+shift+f)"
- **背景**: `rgba(255, 255, 255, 0.1)`

### 收起按钮
- **位置**: 固定在导航栏底部
- **样式**: 使用 `flex-shrink: 0` 和 `margin-top: auto` 实现底部固定

## 技术实现

### 主题配置 (main.tsx)
```typescript
Menu: {
  darkItemBg: 'rgba(0, 0, 0, 0.85)',
  darkSubMenuItemBg: 'rgba(0, 0, 0, 0.85)',
  darkItemSelectedBg: '#5c5c5c',
  darkItemColor: '#fff',
  itemSelectedBg: 'rgba(255, 255, 255, 0.08)',
  itemSelectedColor: '#fff',
  itemHoverBg: 'rgba(255, 255, 255, 0.1)',
  subMenuItemBg: 'transparent',
}
```

### 关键样式类
- `.app-sider`: 导航栏容器
- `.sider-menu`: 菜单容器
- `.sider-menu .ant-menu-item-selected`: 选中项样式
- `.sider-menu .ant-menu-submenu .ant-menu-item-selected`: 二级菜单选中项样式

## 注意事项

1. **不要修改**导航栏的背景色、选中效果、文字颜色等核心样式
2. **不要修改**菜单结构和顺序
3. **不要修改**Logo区域的大小和文字
4. 如需调整，请先更新本文档
